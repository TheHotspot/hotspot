Django Hotspot API v1
HTML only, no JSON yet

URLS:

/                ->      this page (docs)

hotspots/        ->      full listing of hotspots
hotspot/3/       ->      details for hotspot with id=3
hotspot/thingy/  ->      details for hotspot with name="thingy"

users/           ->      full listing of users
user/            ->      details for currently logged in user
user/2/          ->      details for user with id=2
user/blake/      ->      details for user with username="blake"


example access: localhost:8000/api/hotspot/3/
